# ilktoken-contract
Solidity contract for inlock (ILK) token. Details on https://inlock.io/

Standard ERC20 token implementation with the following structure: <br>
 [1] Token [Proxy] --> Token [Library] --> Token [database]
 [2] ICO contract
 [3] Token multisig contract
